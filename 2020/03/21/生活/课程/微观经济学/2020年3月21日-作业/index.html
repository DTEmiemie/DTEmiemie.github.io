弧弹性公式: $e = - \frac{ \frac{ \Delta Q }{ Q } }{ \frac{ \Delta P }{ P } }$
点弹性公式: $ e_d
= \lim_{ \Delta P \to 0 } - \frac{ \Delta Q }{ \Delta P } \times \frac{P}{Q}
= - \frac{dQ}{dP} \times \frac{P}{Q} $

(1)、

由于

​ $P_1=2$ 时，$Q_1 = 500 - 100 \times 2 = 300$

​ $P_2 = 4$ 时，$Q_2 = 500 - 100 \times 4 = 100$

可得弧弹性为 $ \quad e = - \frac{ \frac{ \Delta Q }{ Q } }{ \frac{ \Delta P }{ P } }
= - \frac{ \frac{ -200 }{ 400} }{ \frac{ 2 }{ 6 } }
= - \frac{- \frac{ 1 }{ 2 } }{ \frac{ 1 }{ 3 } } = \frac{ 3 }{ 2 } $

(2)、
